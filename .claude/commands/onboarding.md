# Project Onboarding

Welcome a new developer to this project by explaining its architecture, conventions, and key files.

## Usage
- `/onboarding` — full project walkthrough

## Behavior

Delegate to the `onboarding` agent, which will:

1. Read CLAUDE.md, README, and project structure
2. Explain the architecture and tech stack
3. Walk through key directories and their purpose
4. Highlight coding conventions and important rules
5. Surface known gotchas from `docs/decisions/gotchas.md`
6. List recent architectural decisions from `docs/decisions/`
7. Suggest good first tasks if task tracking is configured
8. Invite questions — stay interactive after the walkthrough
